Detailed Description

Represents the arguments of a choice. The argument has a name, a type and a value. The name and type is given by the SetChoices callback, but it should be possible to update the value using the implemented UI. Before updating the value, it should be checked, that the new value has the correct type. When GetChoices is callsd, the updated values should be used in the result.
