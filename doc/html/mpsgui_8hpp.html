Detailed Description

This file holds a list of method declarations, which defines the API for UI implementations. The API is very simple and consists of callback functions for registering sessions, choices, and receiving the decissions. Many UIs exist simultaneously, and a UI is identified the session it is in, and the participant id (pid) in that session.
